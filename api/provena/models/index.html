    ContextEntry(content_hash, source, source_name, timestamp, provenance=None, metadata=dict(), content_type='str', truncated=False) dataclass ¶

    Immutable representation of a single context input with its content hash.

    Attributes:

    Name Type Description
    content_hash str

    SHA-256 hex digest of the content bytes.
    source ContextSource

    The type of source that produced this context.
    source_name str

    Human-readable name identifying the specific source.
    timestamp datetime

    When this entry was recorded.
    provenance ProvenanceMetadata | None

    Optional origin metadata for validation.
    metadata dict[str, Any]

    Arbitrary key-value pairs attached to the entry.
    content_type str

    The original content type ("str", "bytes", or "json").
    truncated bool

    Whether the content was truncated to fit max_content_bytes.
